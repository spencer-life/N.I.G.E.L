# SPARK Knowledge Base

This folder contains structured SPARK/Chase Hughes content for RAG ingestion.

## How to Add Content

1. Copy output from NotebookLLM
2. Paste into the appropriate `.md` file below
3. Save the file
4. Run `npm run ingest` to process into the database
